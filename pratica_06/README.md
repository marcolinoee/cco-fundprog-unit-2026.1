# Lista de Exercícios - POO em Python

Este repositório contém exercícios práticos sobre Programação Orientada a Objetos em Python.

1. Crie uma classe Estudante com atributos nome e matrícula.
2. Adicione um atributo lista de notas na classe Estudante.
3. Implemente o método adicionar_nota(nota) na classe Estudante.
4. Implemente o método calcular_media() que retorna a média das notas.
5. Crie vários objetos Estudante e teste o cálculo de média.
6. Implemente o método situacao() que retorna 'Aprovado' ou 'Recuperação'.
7. Adicione validação no método adicionar_nota para aceitar apenas valores entre 0 e 10.
8. Crie uma classe Produto com atributo de classe imposto_padrao.
9. Implemente o método preco_com_imposto() na classe Produto.
10. Crie uma classe Turma que armazene uma lista de objetos Estudante.
11. Implemente o método matricular(estudante) na classe Turma.
12. Implemente o método media_geral() na classe Turma.
13. Percorra a lista de estudantes e imprima nome, média e situação de cada.
14. Corrija um erro onde atributos não utilizam self corretamente.
15. Explique a diferença entre atributo de classe e atributo de instância com exemplo em código.
