# 🐸 Simulador de Planejamento e Elisão Tributária
**Grupo Memória Rã** · 1º Período · Python
 
---
 
## O que é?
Programa em Python que recebe os dados de uma empresa e calcula qual regime tributário é o mais barato — **Simples Nacional**, **Lucro Presumido** ou **Lucro Real**.
 
---
 
## Como rodar
```bash
git clone https://github.com/seu-usuario/simulador-tributario.git
cd simulador-tributario
python main.py
```
